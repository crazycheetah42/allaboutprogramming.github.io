How fun programming is and why you should try it

HTML

HTML is basically the whole structure of webpages and JavaScript language can be added using the script tag. Learning HTML is super important, but there are yet LOTS of other languages that there to learn. Also, you can use CSS to make your HTML stylish.

JavaScript

JavaScript is the most popular language there is, and it's not too hard to learn. Just like you end a tag with a closing tag, you must end all JavaScript commands with a semicolon (;). JavaScript is also very fun to learn.

A note about programming

This note is especially for those who expect things to happen first time: You have to be patient, because there might be a mistake, or bug, that you have to fix. It's PERFECTLY okay if things don't work out the first few tries. Just improve your code, and the webpage too will improve!
